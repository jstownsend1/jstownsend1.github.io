Bailey's Used Cars Site

I designed and built this web application from the ground up to help a used car company in Rodgersville, AL attract and sell to more customers and streamline the work it takes to manage the site. I utilized Angular, Firebase realtime database, and Firebase hosting to make this happen.

This video is a more in-depth view of some of the cool features that I built into this app to help the business run more efficiently. The site is hosted here.
